This module is a solution to the problem of getting a "reverse curve" at the foot of each tab on a tabbed panel. I have examined a fairly large number of samples and found no example of this style. Sadly, as developers, we are often stuck with a shop or project requirement to use a "standard" UI module, such as JQuery UI, or Bootstrap UI, to implement such items as menues, layouts managers, panels, and tabbed-panels. Also, the front-end designer may not be conversant, or indifferent, to the implementation of any number of UI modules, or may not even be aware of the contraints posed by CSS.

The first obstacle is to remove all of the cruft generated by whatever tabs module we are stuck with. In this example, we are using the JQuery UI library. For the sake of flexibility, let's separate out a 'tabs-reset' css module (tabs.reset.jqueryui.css).

Because of the specitivity the extant libary may be imposing on the "tabby" elements, our reset-module will sometimes "overspecify" certain selectors. For example, CSSLint will complain the following selector is "too specific", .ui-tabs-nav li.ui-state-default

and could be replaced by: .ui-state-default

The effect of following this advice would be of course to affect every JQuery UI element on the the site, which is probably a broader scope of action then we intended.

For another project, it may be the Bootstrap UI library that is implementing "tabs".
